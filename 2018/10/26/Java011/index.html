String类常用方法–字符与字符串、字节与字符串、字符串比较、查找、替换、截取、拆分、其他操作
。
String的方法必须会背。

字符与字符串

  1. 构造方法
  • public String(Char[] value) 将字符数组变为String类对象
  • public String(char[] value,int offset,int count)将部分字符数组变为String
  1. 普通方法
  • public char charAt(int index) 返回指定索引对应的字符信息
  • public char[] toCharArray() 将字符串以数组的方式返回

字节与字符串

字节使用byte描述，讲座字符串变为字节数组的操作，目的就是为了传输以及编码转换。

  1. 构造方法
  • public String(byte[] bytes) 将全部字节数组变为字符串
  • public String(byte[]bytes,int offset,int length) 将部分字节数组变为字符串
  1. 普通方法
  • public byte[] getBytes() 将字符串变为字节数组
  • public bytes[] getBytes(String charsetName)throws UnsupportedEncodingException 进行编码转换
    在以后讲解IO操作的时候牵扯到字符数组的操作。

字符串的比较

普通方法

  1. public boolean equals(String anObject)
    进行相等判断，区分大小写
  2. public boolean equalsIgnoreCase(String anotherString)
    进行相等判断，不区分大小写
  3. public int compareTo(String anotherString)
    判断两个字符串的大小（按照字符编码比较）返回值有三种结果：
  • =0：表示要比较的两个字符串内容相等
  • >0: 表示大于的结果
  • <0: 表示小于的结果
    现在只有String类的对象才具有大小的关系判断。

字符串的查找

在完整字符串之中要判断某一个子字符串是否存在。
普通方法：

  1. public boolean contains(String s)
    判断指定的内容是否存在
  2. public int indexOf(String str)
    由前向后查找指定字符串的位置，如果查找到了则返回（第一个字母）的索引，如果找不到返回 -1。
  3. public int indexOf(String str,int fromIndex)
    由指定位置从前向后查找指定字符串的位置，找不到就返回 -1。
  4. public int lastIndexOf(String str)
    由后向前查找指定字符串的位置，找不到返回 -1。
  5. public int lastIndexOf(String str,int fromIndex)
    从指定位置由后向前查找字符串的位置，找不到返回 -1。
  6. public boolean startsWith(String predix)
    判断是否以指定的字符串开头
  7. public boolean startsWith(String prefix,int toffset)
    从指定位置开始判断是否以指定的字符串结尾
  8. public boolean endsWith(String suffix)
    判断是否以指定的字符串结尾

使用contains()更加简单，并且在整个Java里面，contains已经成为了查询的代名词。
开头和结尾的判断往往可以作为一些标记在程序之中出现。

字符串替换

使用一个新的字符串替换掉旧的字符串

  1. public String replaceAll(String regex,String replacement)
    用新的内容替换掉全部旧的内容。
  2. public String replaceFirst(String regex,String replacement)
    替换掉首个满足条件的内容。

对于替换，以后还有更加详细的讲解。

字符串截取

普通方法

  1. public String substring(int beginIndex)
    从指定索引截取到结尾
  2. public String substring(int beginIndex,int endIndex)
    截取部分子字符串的数据

字符串拆分

将一个完整的字符串，按照指定的内容拆分为字符串数组。

  1. public String[] split(String regex)
    按照指定的字符串进行全部拆分。
  2. public String[] split(String regex,int limit)
    按照指定的字符串进行部分拆分，最后的数组长度就是由limit决定（如果能拆分的结果很多，数组长度才会由limit决定)的，即：前面拆，后面不拆。

如果是一些敏感字符（正则标记）严格来讲是拆分不了的，如果真的遇见了拆分不了的情况那么使用“\（就是)”进行转义后才可以拆分。
在实际的开发之中，拆分的操作非常常见，因为很多时候会传递一组数据到程序之中进行处理。

其他方法

  1. public String concat（String str)
    字符串连接，与“+”类似
  2. public String toLowerCase()
    转小写
  3. public String toUpperCase()
    转大写
    所有的非字母不会进行任何转换操作
  4. public String trim()
    一般在用户进行输入的时候有可能会携带有无用的空格，我们就要消掉空格。
  5. public int length()
    取得字符串长度
    某些情况下要求用户输入的数据长度是有限制的，可以用此方式判断。数组也有length,它们的形式不同：
  • 数组对象.length
  • String对象.length()
  1. public String intern()
    数据入池
  2. public boolean isEmpty()
    判断是否是空字符串（不是null，而是””，长度是 0）
    如果觉得isEmpty()不方便可以使用“””.equals(str)”。

String提供了大量的方法，却少了一个重要的方法–initcap()功能，首字母大写，其余字母小写，只能自己实现。虽然Java的类库没有此功能，但是第三方的组件包里此内容。

总结

所有讲解过的方法的要求：记下方法名称、方法作用、参数的类型及个数、返回值类型。必须要背下来，否则写代码时会有很大的挫折感。
